Easy Time Picker Widget - Simplify your Time fields

Simplify Time Entry with a User-Friendly, Customizable, and Responsive Widget.

Accurate Time Tracking

Easily pick and input time values with a user-friendly interface, reducing the chances of errors.

Responsive Design: Optimized for both desktop and mobile devices, providing a consistent user experience across all platforms.

Seamlessly integrate the widget into your Odoo forms and views.

Usage

To use the widget in a form, just include: 

<field name="field_name" widget="easy_timepicker"/>

Example 
<field name="duration_expected" widget="easy_timepicker"/>

Note: This widget only works in Float fields
